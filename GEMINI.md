# Spazio Backend - Core Workflows

## Skill Activation Rule
Whenever working on the backend, the following expert skills **must be automatically activated** to ensure architectural consistency, rigorous testing, and accurate API documentation:

- `go-vertical-slice`: Enforces the handler/service/repository vertical slicing pattern.
- `golang-testing`: Enforces the use of the native Go testing library and table-driven tests (no external assertion libraries).
- `golang-pro`: Enforces idiomatic Go 1.21+ patterns, concurrency, and performance optimization.
- `swagger-docs`: Enforces accurate OpenAPI annotations for Gin handlers.
